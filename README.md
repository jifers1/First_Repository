 - Step 1 **Creating a Local Repository**:
 Navigate to your project directory
 ```

    cd my_app
 ```
- Step 2 **Initialize a new Git repository**
 
 ```
    git init
```
-Step 3 **Adding files to the Repository**
```
    git add file_name
         or
    git add .
    // the '.' refers to the current directory
```
